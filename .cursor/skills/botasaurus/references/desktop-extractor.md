# Desktop Extractor

A Desktop Extractor is a standalone application that runs on Mac, Windows, and Linux. It extracts data from websites, PDFs, Excel files, and other documents.

## Advantages Over Web-Based Scrapers

- **Zero Infrastructure Costs**: Runs on user's machine, no cloud computing fees. Lower costs → lower pricing → more customers.
- **Faster Execution**: Instant execution, no cloud resource allocation delays. Uses user's system (faster than shared cloud servers).
- **Increased Engagement**: App on desktop encourages frequent use vs. web tools users must actively visit.
- **Cross-Platform in 1 Day**: With Botasaurus, launch for Windows, macOS, and Linux in one day.

## Built-in Features

Botasaurus Desktop includes:
- Task management
- Data Table
- Data export (Excel, CSV, etc.)
- Sorting & Filtering
- Caching
- And more

## Market Opportunity

Document Data Extraction is a large untapped market. Example: Accountants spend hundreds of hours manually entering invoice data for tax filings. A desktop extractor transforms this into a minutes-long task with 100% accuracy.
